Generate Completion

Send a prompt to the active neural core and receive a streamed response.

ParameterDescription
model
string		 ID of the model to use (e.g., "llama-3-8b").
messages
array		 List of message objects. 
curl https://api.luca.local/v1/cortex/chat \
  -X POST \
  -H "Authorization: Bearer " \
  -d '{
    "model": "llama-3-8b",
    "messages": [
      {"role": "user", "content": "Hello world"}
    ]
  }'
POST

Store Memory

Inject a new semantic memory into the persistent vector graph.

curl https://api.luca.local/v1/memory/store \
  -X POST \
  -d '{
    "content": "User prefers dark mode",
    "tags": ["ui", "preference"]
  }'
